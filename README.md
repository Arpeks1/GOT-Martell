# 🐉 Sitio de GOT sobre la casa Martell🐉
- Descripción

 Sitio web sobre la casa de Martell de la serie Game Of Thrones, el proyecto busca mostrar los rasgos destacables de la casa Martell, en conjunto con un diseño que haga recordar a la esencia misma acerca de esta querida casa. 

---

## 🛠️ Tecnologías utilizadas

- HTML: FrontEnd, Estructura.
- CSS: FrontEnd, Diseño.
- JavaScript: BackEnd, Lógica.
- Bootstrap: Framework.
